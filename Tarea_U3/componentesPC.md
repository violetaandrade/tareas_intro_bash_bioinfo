## **7) Reflexión final (breve)**

Mi equipo computacional representa una laptop de gama media orientada principalmente al uso de gaming ligero. Si bien es adecuada para tareas computacionales moderadas, presenta limitaciones importantes para flujos de trabajo exigentes en bioinformática con gran volumen de datos. 
Principalmente la memoria RAM podría aumentarse a los 16GB que permite el sistema. Con este aumento podría ser sucifiente para proyectos pequeños o medianos. 
Además, la capacidad de almacenamiento es restrictiva para manejar un alto volumen de datos. 
En conclusión es suficiente para tareas estándar, pero limitado para análisis intensivos y simulaciones extensas.
